# Embedded Vision AI on ESP32: Building Real-Time Edge Intelligence with Microcontrollers

**Subtitle:** Build a smart camera powered by embedded AI in just a few hours — deploy a real computer vision model on an ESP32 device.

---

## Overview

Artificial intelligence is rapidly moving from massive cloud servers into billions of small embedded devices. Cameras, sensors, robots, and IoT systems are increasingly expected to make intelligent decisions locally — without sending data to the cloud.

In this hands-on workshop, participants build a real embedded computer vision system using an ESP32 microcontroller and a camera module. Instead of relying on GPUs or cloud APIs, they deploy a lightweight neural network directly onto the device and run real-time inference on live camera frames.

The workshop covers the complete pipeline — from understanding embedded AI constraints to deploying a working AI-powered smart camera. By the end, the ESP32 device classifies objects or detects events and triggers actions based on what it sees.

This session is designed for developers who want to understand how AI actually runs on constrained hardware — the same techniques used in modern robotics, smart sensors, and edge computing platforms.

## Why Now

Edge AI is one of the fastest-growing areas in technology. As AI expands beyond data centers into billions of connected devices, developers increasingly need to understand how machine learning runs on constrained hardware.

Microcontrollers like the ESP32 are now powerful enough to run optimized neural networks locally, enabling intelligent cameras, robotics systems, and smart sensors that operate without cloud dependencies.

Learning to deploy AI directly on embedded hardware is quickly becoming a valuable skill for developers working in IoT, robotics, and edge computing.
